# Viva Mais 🌿

Portal digital focado no bem-estar da terceira idade, desenvolvido como projeto da Atividade Extensionista II do curso de Análise e Desenvolvimento de Sistemas – UNINTER.

## Sobre o Projeto

O Viva Mais é um portal web criado para facilitar o acesso de idosos a informações de saúde e bem-estar na região de São José dos Pinhais/PR. O layout foi desenvolvido com foco em alta legibilidade e acessibilidade, aplicando princípios de Desenho Universal ("Informação de Fácil Percepção" e "Baixo Esforço Físico").

**ODS vinculada:** 03 – Saúde e Bem-estar

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Acessibilidade

- Fontes e botões em tamanhos ampliados
- Contraste adequado para usuários com limitações visuais
- Navegação simplificada para baixo esforço físico e cognitivo
